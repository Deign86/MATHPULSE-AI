# MathPulse Grade 11-12 Dataset Pack

This dataset pack supports evaluation and synthetic generation workflows for MathPulse AI.

## Splits

- eval: Curated Grade 11-12 benchmark prompts with expected answers.
- synthetic: Model-generated variants for stress testing.
- train_like_anonymized: Optional anonymized, non-identifying examples.

## Data safety

All records are synthetic or anonymized. Student-identifying data is intentionally excluded.
