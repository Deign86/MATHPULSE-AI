# DepEdMathTutor-Qwen2.5-7B

Specialized for Philippine DepEd Senior High School Grades 11-12 Mathematics.

## Features
- DepEd curriculum aligned: General Math, Business Math, Statistics and Probability, Precalculus
- Step-by-step solutions in a fixed format with verification
- Long-context handling for module excerpts and table-based questions
- QLoRA training flow for Kaggle T4/P100 and Hub publishing
- 8k context window target for long educational inputs

## Usage
```python
from transformers import AutoModelForCausalLM, AutoTokenizer

model = AutoModelForCausalLM.from_pretrained("yourusername/deped-math-qwen2.5-7b-merged")
tokenizer = AutoTokenizer.from_pretrained("yourusername/deped-math-qwen2.5-7b-merged")
```

## Training Details
- Base: Qwen2.5-7B-Instruct
- Method: QLoRA (4bit adapters, optional 16bit merged export)
- Dataset: DepEd SHS math prompts in JSONL messages format
- Hardware: Kaggle T4/P100

## File Map
- `system_prompt.txt`: fixed training system prompt
- `train.jsonl`: sample dataset line in required chat schema
- `deped-math-qlora-training.ipynb`: Kaggle notebook generated by `scripts/generate_colab_qwen25_notebook.py`
- `upload_to_hf.py`: publish LoRA and merged weights to Hugging Face Hub
- `generate_data.py`: synthetic prompt generator scaffold

## Kaggle Commands
```bash
# Create a Kaggle dataset folder with train.jsonl inside
mkdir -p /kaggle/working/deped-math-sft
cp train.jsonl /kaggle/working/deped-math-sft/train.jsonl
cd /kaggle/working/deped-math-sft
kaggle datasets init -p .
# Edit dataset-metadata.json as needed, then create:
kaggle datasets create -p . --dir-mode zip

# Push notebook kernel bundle from repo root
kaggle kernels push -p .kaggle-kernels/<your-kernel-slug>
```

## Pipeline Execution
1. Upload `train.jsonl` as a Kaggle dataset.
2. Run notebook to produce `deped-math-lora` and `deped-math-7b-merged` artifacts.
3. Download artifacts or sync to working directory.
4. Login: `huggingface-cli login`.
5. Publish: `python training/deped_qwen25_pack/upload_to_hf.py --repo-id yourusername/deped-math-qwen2.5-7b`.
